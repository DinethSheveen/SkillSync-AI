### Prerequisites

* Node.js (v18 or later recommended)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/DinethSheveen/SkillSync-AI.git
```

2. Install dependencies for both client and server:

```bash
cd SkillSync-AI
npm install
```

3. Start the development servers:

```bash
npm run dev
```
